3. 无私募
4. 无预挖

在技术上实现了：

1. 完全由智能合约实现，公开、透明、可验证
2. 发行代币通过智能合约全部进入去中心化交易所的资金池，永远无法移除

它在简单性上做到了极致：只有买和卖
